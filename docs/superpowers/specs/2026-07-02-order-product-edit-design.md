# 订单产品编辑功能设计规格

**日期**: 2026-07-02  
**主题**: 编辑订单产品 Dialog、表格内联编辑、右键菜单与采购/库存联动  
**相关文件**:
- `client/widgets/order_summary/order_detail_panel.py`
- `client/widgets/product_item_edit_dialog.py`（修改现有编辑订单产品 Dialog）
- `client/main.py`
- `backend/crud/pi.py`
- `backend/routers/pi.py`

---

## 1. 目标

让订单详情中的产品项支持灵活编辑：
- 简单数字/文本字段（报价、预付款、尾款、运费、杂费）在表格内联编辑，失焦自动保存。
- 双击列 2-9、交货日期、包装规格相关列打开"编辑产品"Dialog。
- "编辑产品"Dialog 显示所有字段：负责编辑的字段可编辑，采购/供应商字段锁定显示并提供入口按钮。
- 右键菜单识别当前列，打开 Dialog 后高亮对应字段。
- 采购后锁定产品基础信息、包装规格；但允许通过"更换供应商"Dialog 更换工厂/店铺并重新生成采购单。
- 转为正式 PI 后，产品基本信息字段锁定不可编辑。

## 2. 字段编辑方式

### 2.1 表格内联编辑（失焦自动保存）

| 表格列 | 表头 | 字段名 | 控件 |
|---|---|---|---|
| 10 | 报价(USD/RMB) | `unit_price` | 数字输入 |
| 13 | 客户预付款 | `customer_prepayment` | 数字输入 |
| 14 | 待收尾款 | `remaining_payment` | 数字输入 |
| 18 | 运费 | `shipping_fee` | 数字输入 |
| 19 | 杂费 | `misc_fee` | 数字输入 |

- 仅这 5 列设置 `Qt.ItemIsEditable` 标志。
- 双击这 5 列单元格进入编辑，失去焦点时调用 `PUT /pi/items/{item_id}`。
- 双击列 2-9、23、29/30/31/33/34/37 时，打开"编辑产品"Dialog。
- 只提交变更的字段，失败时弹窗提示并回滚显示。

### 2.2 编辑订单产品 Dialog（右键/双击打开）

**修改 `client/widgets/product_item_edit_dialog.py` 中的 `ProductItemEditDialog`**，在现有 6 个字段基础上扩展为完整的订单产品编辑 Dialog，初始尺寸调整为 **768 × 1280**。

现有 Area 与新增字段：

| 现有 Area | 已有字段 | 本次新增字段 |
|---|---|---|
| 产品信息 | 产品名称、OE号、数量、单价、币种、客户需求备注 | 客户产品编号（col 2）、图片（col 6）、客户型号（col 7）、产品特性（col 8） |
| 数量与日期 | - | 交货日期（col 23） |
| 包装规格 | 包装方式、件数设置、采购选项、纸箱尺寸、体积、重量 | 无 |
| 付款信息 | 客户预付款、待收尾款、工厂订金、工厂尾款 | 运费（col 18）、杂费（col 19） |
| 采购/供应商 | - | 新增 Area，锁定显示：工厂简称、店铺链接、工厂编号、品牌、采购价格、开票情况 |
| 客户回复 | 最新回复预览、打开回复管理 | 无 |

#### 锁定规则（在 Dialog 内根据状态禁用/启用）

- 未采购且未转正式：全部可编辑字段可编辑。
- 已转正式 PI：产品基本信息字段（客户产品编号、OE号、产品名称、客户型号、产品特性、数量）锁定。
- 已采购：产品基本信息、交货日期、包装规格字段锁定；财务字段（报价、预付款、尾款、运费、杂费）可编辑。

#### 采购/供应商 Area 按钮

- **采购 Dialog**：打开现有采购流程 Dialog。
- **更换供应商 Dialog**：
  - 无采购单时禁用，tooltip 提示"尚未生成采购单"。
  - 有采购单时点击弹出第一次确认；进入更换供应商 Dialog 后保存时弹出第二次确认；后端执行前进行最终校验（第三次确认/强校验）。

#### 高亮规则

- 无论从右键菜单还是双击单元格进入 Dialog，均根据触发列号定位并高亮对应字段。
- 若触发列属于编辑订单产品 Dialog 范围（列 2-9、10、13、14、18、19、23、29-31、33-34、37），则打开 Dialog 并高亮该列对应的输入控件。
- 若触发列属于采购/供应商范围（列 17、21、22、25、26、32、39、40），则打开 Dialog 并自动滚动到采购/供应商 Area，同时高亮对应锁定字段。

### 2.3 更换供应商 Dialog（右键"更换供应商"）

| 分组 | 表格列 | 表头 | 字段名 | 控件 |
|---|---|---|---|---|
| 供应商信息 | 21 | 工厂简称 | `factory_short_name` / `supplier_name` | 文本 |
| | 22 | 店铺链接 | `supplier_url` / `shop_url` | 文本 |
| | 32 | 工厂编号 | `factory_code` | 文本 |
| | 39 | 品牌 | `brand` | 文本 |
| 采购价格 | 17 | 采购价格 | `purchase_price` / `factory_price` | 数字 |
| | 25 | 工厂订金 | `factory_deposit` | 数字 |
| | 26 | 工厂尾款 | `factory_balance` | 数字 |
| 开票 | 40 | 开票情况 | `invoice_status` / `invoice_type` | 下拉 |

- 右键采购/工厂/供应商相关列时，打开此 Dialog 并高亮对应字段。
- 保存后重新生成采购单（见 3.2 节）。

### 2.4 不编辑字段

| 类型 | 列 | 说明 |
|---|---|---|
| 订单级 | 0 订单日期、1 ORDER NO. | 随 PI 主单编辑 |
| 派生 | 11 合计金额、15 预估美金报价、16 预估毛利率、20 总金额、35 箱数、36 预估体积、38 总重量 | 由其他字段计算 |
| 流程状态 | 24 是否已收货、27 入库操作、28 入库数量 | 由采购/入库流程写入 |

## 3. 右键菜单

在现有右键菜单基础上增加两项：

```
采购该产品
重新采购
─────────
入库该产品
删除商品
─────────
编辑产品                ← 新增
更换供应商              ← 新增
采购快照                ← 新增
访问店铺网站            ← 新增
─────────
缺货标记
```

- 点击"编辑产品"/"更换供应商"/"采购快照"/"访问店铺网站"时记录当前右键列号。
- "采购快照"：打开只读浮窗/Dialog，展示该产品项当前采购状态快照，包括采购价格、工厂简称、店铺链接、工厂订金/尾款、开票情况、采购单状态、已采购数量等。
- "访问店铺网站"：使用 `QDesktopServices.openUrl` 打开列 22 店铺链接；若链接为空，弹出提示"店铺链接为空"。
- 根据列号决定打开哪个 Dialog 以及高亮哪个字段。

## 4. 锁定规则

### 4.1 状态定义

- `is_formal`：PI 已保存为正式记录（"保存正式纪录"后）。
- `is_purchased`：该产品项已生成采购单。
- `has_purchase_order`：存在关联的采购单记录。

### 4.2 未采购且未转正式时

- 编辑订单产品 Dialog：全部字段可编辑。
- 更换供应商 Dialog：全部字段可编辑。

### 4.3 已转正式 PI 后

- 编辑订单产品 Dialog：
  - 锁定：产品基本信息字段（列 2-9：客户产品编号、OE号、备注、产品名称、图片、客户型号、产品特性、数量）。
  - 可编辑：财务字段（10/13/14/18/19）、交货日期（23）、包装规格（29/30/31/33/34/37）、客户回复（12）。
- 更换供应商 Dialog：
  - 全部字段可编辑，保存后重新生成采购单。

### 4.4 已采购后

- 编辑订单产品 Dialog：
  - 锁定：列 2-9、23 交货日期、包装规格字段（29/30/31/33/34/37）。
  - 可编辑：财务字段（10/13/14/18/19）、客户回复（12）。
- 更换供应商 Dialog：
  - 全部字段解锁，允许更换工厂/店铺/价格/开票信息。
  - 保存后重新生成采购单。

## 5. 保存与联动

### 5.1 后端 API

继续复用 `PUT /pi/items/{item_id}`，由 `update_pi_item` 处理字段更新。

### 5.2 编辑订单产品 Dialog 保存

- 修改 `ProductItemEditDialog` 的 `_save` 方法，在原有 6 个字段基础上新增：客户产品编号、客户型号、产品特性、图片 URL、交货日期、包装规格、运费、杂费。
- `quantity` / `unit_price` 变化时，后端自动：
  - 重算 `total_price`。
  - 刷新 PI 主单 `total_amount`。
- `quantity` 减少时，后端检查 `InvInventory`：
  - 若库存数量 > 新数量，返回 400 并提示"请先处理库存"，前端保持原值。
  - 若库存数量 ≤ 新数量，更新库存数量为新的订单数量（或标记待核对，视业务决定）。

### 5.3 更换供应商 Dialog 保存

- 提交供应商相关字段。
- 确认流程（共三次）：
  1. 在编辑订单产品 Dialog 中点击"更换供应商"Button 时，弹出第一次确认："确定要更换供应商吗？当前采购单将被取消并重新生成。"
  2. 在更换供应商 Dialog 中点击"保存"时，弹出第二次确认："确定要按以上信息更换供应商并重新生成采购单吗？"
  3. 后端执行重新生成前，进行最终校验（第三次确认或强校验）：检查采购单是否已收货/已入库；若是，返回 400 拒绝；否则执行删除旧采购单并创建新采购单。
- 后端逻辑：
  1. 更新 PI item 的供应商字段。
  2. 查找该产品项关联的采购单（`PoPurchaseOrderItem`）。
  3. 若采购单状态为"未下单"/"已下单未收货"：
     - 删除原采购单及采购项，根据新供应商信息重新创建采购单。
  4. 若采购单已收货/已入库：
     - 返回 400，提示"已收货/入库，无法更换供应商，请先处理库存"。
  5. 更新库存关联记录的供应商信息（如果有）。

### 5.4 表格内联保存

- 失焦时立即调用 `PUT /pi/items/{item_id}`，只传修改字段。
- 保存成功后刷新当前行显示；失败时回滚并提示。

## 6. UI/UX 细节

### 6.1 Dialog 高亮

- 打开 Dialog 时，根据传入的 `focus_field` 或 `focus_column`：
  - 滚动到对应分组。
  - 对输入框设置黄色边框背景 2 秒。
  - 如果是下拉框，展开下拉列表。

### 6.2 图片上传

- 编辑订单产品 Dialog 中图片区域显示当前图片缩略图。
- 点击"上传图片"调用现有图片上传接口，成功后更新 `image_url`。
- 点击"清除图片"移除图片（后端字段置空）。

### 6.3 日期选择器

- 交货日期使用 `QDateEdit`，格式 `yyyy-MM-dd`。

### 6.4 下拉选项

- 包装方式：`["1件/箱", "多件/箱", "1件多箱"]`
- 开票情况：`["增票", "普票", "不开票"]`
